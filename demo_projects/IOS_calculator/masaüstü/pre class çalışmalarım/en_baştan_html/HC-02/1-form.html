< input type="password"> Girilenler * halinde görülür
< input type="radio"> Bir radyo düğmesi görüntüler (birçok seçenekten birini seçmek için) value niteliği, seçilen için sunucuya gönderilen değeri belirtir
< input type="checkbox"> Bir onay kutusu görüntüler (birçok seçenekten sıfır veya daha fazlasını seçmek için)
< input type="submit"> Bir gönder düğmesi görüntüler (formu göndermek için) value niteliği bir düğmenin üzerinde görünecek metni belirtmek için kullanılır
< input type="button"> Tıklanabilir bir düğme görüntüler form verilerini sunucuya göndermek için bir düğme tanımlamak için kullanılır
